# Núcleo Acadêmico

## ## 🚧 Status do Projeto

O projeto encontra-se em fase de desenvolvimento e algumas funcionalidades ainda estão sendo implementadas.

## 📋 Funcionalidades Planejadas

* Sistema de cadastro e login de usuários;
* Cadastro e gerenciamento de disciplinas;
* Cadastro e visualização de atividades acadêmicas;
* Área para consulta de materiais de estudo;
* Organização de atividades por disciplina;
* Perfil do usuário;
* Painel inicial com informações acadêmicas;
* Interface responsiva para dispositivos móveis e desktop.

## 🔮 Melhorias Futuras

* Calendário acadêmico;
* Sistema de notificações;
* Upload de materiais de estudo;
* Controle de prazos de atividades;
* Área para avisos e comunicados.
